git

pwd - shows the current directory

```sh
pwd
```

ls - show the contents of the directory
```
ls
```

cd - change the directory to the specified path

if the computer got a second drive and its name is E:, we can go to it with cd
```sh
cd E:
```

mkdir - create a new directory with the specified name
```sh
mkdir my-programming-notes
```

touch - create a new file with the specified name

```sh
touch my-note1.txt
```
